# BI Analytics Pipeline — Commercial Real Estate

An end-to-end data pipeline and analytics project built to demonstrate skills for a Business Intelligence Analyst role at JLL.

Pulls commercial real estate market data from public sources, transforms it in Snowflake via dbt, and surfaces insights through an interactive Streamlit dashboard and a Claude Code-queryable knowledge base.
